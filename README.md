## Страница «Главная»

Использует баблиотеки:
#json, #requests, #API, #datetime, #logging, #pytest, #pandas

Реализован набор функций и главная функция, принимающая на вход строку с датой и временем в формате 
YYYY-MM-DD HH:MM:SS
и возвращающую JSON-ответ со следующими данными:

Приветствие в формате "???", где ???:

 — «Доброе утро» / «Добрый день» / «Добрый вечер» / «Доброй ночи», 
в зависимости от текущего времени.

По каждой карте:
- последние 4 цифры карты;
- общая сумма расходов;
- кешбэк (1 рубль на каждые 100 рублей);
- топ-5 транзакций по сумме платежа;
- курс валют;
- стоимость акций из S&P500.
